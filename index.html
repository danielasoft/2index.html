HTML entities are used to represent special characters that have a specific meaning in html, we can use HTML entities in the following situations:
  1. Reserved Characters: Use entities for characters that have special meaning in html.such as <,>, & and ". Example:

    10 < 20

  2. Non-Breaking Space: Use   to insert a non-breaking space, which prevents the browser from breaking the line at the point. example:

    First Name

  3. Symbols: Use entities for special symbols that are not directly typable on a keyboard, such as copyright symbol. example:

    © 2024

  5. Compatibility: Use entities for characters that might be interpretd incorrectly by some browsers or when you want to ensure maximum Compatibility. example:

    10 × 5=50
